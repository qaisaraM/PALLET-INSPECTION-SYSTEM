Detection Pipeline

Multi-Layer
Vision System

Production environments are messy. Lighting changes. Parts reflect. Slots get occluded. Pure YOLO isn't enough. SYSTEM combines AI + rule-based spatial reasoning.

01
Camera Feed
Hikrobot GigE / USB3 Vision industrial camera. Real-time frame acquisition via MVS SDK.
02
YOLOv8 Detection
Custom-trained model detects filled or empty slots. Configurable confidence & IOU thresholds per pallet mode.
03
Grid Gap Inference
Spatial clustering reconstructs pallet grid. Missing cells inferred from tray geometry. Yellow overlays mark missing slots.
04
OpenCV Validation
Hough Circle Transform + Blob Detection. IOU cross-validation with YOLO catches missed parts. Duplicate suppression.
05
Auto Count + Report
Automatic judgement rendered. Screenshot captured. Excel log generated instantly.

Engineering Features

Built for the
Factory Floor

🔋

Power Suspend Recovery

Win32 WM_POWERBROADCAST listener. Camera shuts down gracefully on sleep, reconnects automatically on resume. Zero operator intervention.

🔥

Live Excel Hot Reload

QFileSystemWatcher detects database changes while the app runs. Part codes, operator names — all update in real time with 100ms debounce protection.

↩

Atomic Undo Stack

Every count stores full application state. Undo restores part quantity, totals, LCD counters, and judgement state — atomically.

⌨

Operator-Friendly UI

Touchscreen support, on-screen keyboard, smart autocomplete, large buttons for gloves. Fullscreen operation. Color-coded counters. Built for real humans.

Key Lessons

What I
Learned

AI Alone Is Not Enough

Structured industrial environments benefit massively from hybrid systems. Neural detection + spatial logic + rule validation outperforms any single approach.

Neural + Spatial + Rules

Hardware Integration Is Hard

Industrial cameras introduce driver failures, suspend/resume issues, packet loss, and timing edge cases. Recovery logic matters as much as ML accuracy.

Real Operators Shape Product

Autocomplete, undo, large buttons, touchscreen support, visual feedback — these weren't "nice-to-have". They were essential for real-world deployment.

Geometry Is a Signal

Industrial trays are highly structured. Spatial relationships between slots become a powerful secondary validation layer beyond pure neural detection.
